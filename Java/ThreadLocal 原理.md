# ThreadLocal 原理

## 概述

- 其作用域覆盖线程，而不是某个具体任务
- 其“自然”的生命周期与线程的生命周期“相同”。Thread 类中有 threadlocal 变量

## 使用场景

- 实现单个线程单例以及单个线程上下文信息存储，比如交易id等
- 实现线程安全，非线程安全的对象使用ThreadLocal之后就会变得线程安全，因为每个线程都会有一个对应的实例
- 承载一些线程相关的数据，避免在方法中来回传递参数
