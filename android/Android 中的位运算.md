# 位运算

## 背景

**对于 Java 而言，由于 Java 是跨平台语言，所以 JVM 表现下的基础数据字节长度其实都是一致的。**

```
int：4 个字节。

short：2 个字节。

long：8 个字节。

byte：1 个字节。

float：4 个字节。

double：8 个字节。

char：2 个字节。

boolean：boolean属于布尔类型，在存储的时候不使用字节，仅仅使用 1 位来存储，范围仅仅为0和1，其字面量为true和false。
```

## 运算符

- 与运算（&）: 判断是否包含某个`flag`
- 或运算（|）：`flag`的组合
- 非运算（~）：清除某个`flag`
- 异或运算（^）：相同取0，不同取1
